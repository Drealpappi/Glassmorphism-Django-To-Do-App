# Glassmorphism-Django-To-Do-App
A premium Django task manager with a "frosted glass" UI.  Auth: Secure Login/Register.  Privacy: Each user sees only their own tasks.  CRUD: Full Create, Read, Update, Delete.  Style: Modern Glassmorphism &amp; gradients.
